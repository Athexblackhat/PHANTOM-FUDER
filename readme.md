## 🔧 Troubleshooting
Common Issues
Issue 1: "apktool not found"
bash
# Solution: Install apktool or place in current directory
```
sudo apt-get install apktool  # Linux
```
```
brew install apktool          # macOS
```
# Or download manually:
```
wget https://raw.githubusercontent.com/iBotPeaches/Apktool/master/scripts/linux/apktool
chmod +x apktool
sudo mv apktool /usr/local/bin/
```
Issue 2: "Java not found"

# Solution: Install Java JDK
```
sudo apt-get install openjdk-11-jdk  # Linux
```
```
brew install openjdk@11              # macOS
```
# Windows: Download from Oracle website
Issue 3: "PyQt6 import error"
bash
# Solution: Install PyQt6
```
pip install PyQt6 PyQt6-WebEngine
```
Issue 4: "Build failed - memory error"
bash
# Solution: Increase memory limit or reduce protection level
python APK-FUD-V5.py build app.apk --level 4  # Lower level
# Or increase swap space
sudo fallocate -l 2G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
Issue 5: "APK signing failed"
bash
# Solution: Generate keystore manually
```
keytool -genkey -v -keystore my.keystore -alias mykey \
    -keyalg RSA -keysize 2048 -validity 10000
```
# Then use in build
```
python APK-FUD-V5.py build app.apk --keystore my.keystore --alias mykey
```
